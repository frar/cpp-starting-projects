Detailed Description

Class that defines the Qt gui interface for task scheduler and that launches the timer for starting task scheduling.

For reference on sysTray see the following url or example systray among Qt official examples: http://doc.qt.io/archives/qt-4.8/qt-desktop-systray-example.html
